## 3.1 运算符与表达式：C++ 的“计算骨架”

运算符是用于对数据执行数学或逻辑操作的特殊符号。表达式由运算符和操作数组成，每个表达式都会产生一个值。

### 3.1.1 运算符分类与优先级

C++ 拥有丰富的运算符，按功能可分为：算术、关系、逻辑、位运算、赋值、条件、逗号等。按操作数个数分为：一元（如 `++`）、二元（如 `+`）、三元（仅 `?:`）。

**优先级** 决定了运算顺序。例如 `3 + 4 * 5` 先乘后加得 23。**结合性** 决定同优先级运算符的计算方向，例如 `a = b = 5` 是右结合（从右向左）。

## 3.2 算术运算符：小心整数除法和溢出

### 3.2.1 基本运算

| 运算符 | 含义   | 示例    | 结果 |
|--------|--------|---------|------|
| `+`    | 加法   | `15+2`  | 17   |
| `-`    | 减法   | `2-15`  | -13  |
| `*`    | 乘法   | `15*2`  | 30   |
| `/`    | 除法   | `15/2`  | 7    |
| `%`    | 取余   | `15%2`  | 1    |

**关键陷阱**：
- 整数除法向零取整（C++11 起）：`21 / 5` = 4，`-21 / 5` = -4。
- 取余 `%` 只能用于整数，结果的符号与被除数相同：`21 % -5` = 1。
- 溢出是未定义行为：`short` 最大值 32767 + 1 可能变成负数或崩溃。

### 3.2.2 自增 `++` 与自减 `--`

前置 `++x`：先加1，返回 x 本身（左值）。  
后置 `x++`：先返回原值，再加1（返回右值）。

```cpp
int a = 3;
std::println("{}", a++);   // 输出 3，a 变为 4
std::println("{}", ++a);   // 输出 5
```

**危险示例**（未定义行为）：
```cpp
int x = 1;
int d = x + ++x;   // 结果不确定，不同编译器可能不同
```
**规则**：不要在同一表达式中多次修改同一个变量。

> 完整版会详细讲解左值/右值、连续自增 `++ ++x` 的合法性以及为何 `x++ ++` 会编译错误。

### 3.2.3 数学函数 `<cmath>`

复杂运算（平方根、幂、三角函数）需要 `<cmath>`：

```cpp
#include <cmath>
std::println("{}", sqrt(2.0));   // 1.414...
std::println("{}", pow(2, 10));  // 1024.0
```

## 3.3 位运算：直接操控二进制

位运算符对整数二进制位进行操作，常用于权限控制、高效乘除、加密等。

| 运算符 | 含义     | 示例（8位）                     |
|--------|----------|----------------------------------|
| `&`    | 按位与   | `00100101 & 00010110 = 00000100` |
| `|`    | 按位或   | `00100101 | 00010110 = 00110111` |
| `^`    | 按位异或 | `00100101 ^ 00010110 = 00110011` |
| `~`    | 按位取反 | `~00100101 = 11011010`           |
| `<<`   | 左移     | `00100101 << 2 = 10010100`       |
| `>>`   | 右移     | `00100101 >> 2 = 00001001`       |

**注意**：
- 左移 `n` 位相当于乘以 `2^n`（无符号或正数），但溢出为未定义行为。
- 有符号负数的右移是算术右移（高位补符号位），无符号数是逻辑右移（高位补0）。

**示例**：判断一个数是否是 2 的幂（经典位运算技巧）：
```cpp
bool isPowerOfTwo(unsigned int n) {
    return n && !(n & (n - 1));
}
```

## 3.4 赋值运算符：右结合与左值要求

基本赋值 `=` 和复合赋值（`+=`、`-=`、`*=`、`/=`、`%=`、`&=`、`|=`、`^=`、`<<=`、`>>=`）。

**关键点**：
- 右结合：`a = b = 5` 等价于 `b = 5; a = b;`。
- 左边必须是**左值**（可修改的变量），不能是常量或临时值：`5 = a;` 错误。
- `const` 变量不能出现在赋值左侧。

```cpp
int a, b;
a = b = 5;         // 合法
const int ci = 10;
ci = 20;           // 错误
```

## 3.5 关系运算符：比较与太空船 `<=>`（C++20）

传统比较运算符返回 `bool`：`==`、`!=`、`<`、`<=`、`>`、`>=`。

**浮点数比较陷阱**：由于精度误差，`0.1 + 0.2 == 0.3` 通常为 `false`。正确做法是比较差的绝对值：

```cpp
if (std::fabs(a - b) < 1e-9) { /* 相等 */ }
```

**C++20 太空船运算符 `<=>`**：三路比较，返回比较类别对象（如 `std::strong_ordering`），可同时获得 `<`、`==`、`>` 信息。

```cpp
#include <compare>
int a = 10, b = 20;
auto res = a <=> b;
if (res < 0) std::println("a < b");
else if (res > 0) std::println("a > b");
else std::println("a == b");
```

**优势**：只需重载一个运算符就能自动支持所有比较，尤其适合自定义类型（后续章节）。

## 3.6 逻辑运算符：短路求值

| 运算符 | 含义 | 示例              |
|--------|------|-------------------|
| `&&`   | 与   | `(a>0 && b>0)`    |
| `||`   | 或   | `(a==0 || b==0)`  |
| `!`    | 非   | `!flag`           |

**短路特性**：
- `false && anything` → 不计算 anything
- `true || anything`  → 不计算 anything

利用短路可以写出简洁的保护代码：
```cpp
if (ptr != nullptr && ptr->value > 10) { ... }  // 安全
```

**替代关键字**：`and`（`&&`）、`or`（`||`）、`not`（`!`），使代码更自然。

```cpp
if (x and y) { ... }   // 等价于 x && y
```

## 3.7 特殊运算符：条件 `?:` 与逗号 `,`

- **条件运算符** `cond ? true_expr : false_expr`：三目运算符，可内嵌。
- **逗号运算符** `expr1, expr2`：依次计算 expr1、expr2，返回 expr2 的值。**注意**：变量声明中的逗号是分隔符，不是运算符。

```cpp
int a = 1, b = 2;      // 逗号是分隔符
int c = (a, b);        // 逗号运算符，c = 2
```
