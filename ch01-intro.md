## 1.1 计算机与编程语言 —— 从“做饭”到“指令”

计算机虽然不能“猜意思”，但它能严格按照指令一步一步执行。比如自动售票机：

- 你投币、按按钮（**输入**）
- 机器验证票务、扣款（**CPU 处理**）
- 记住票价、剩余票数（**存储**）
- 打印车票（**输出**）

计算机的指令非常底层，例如 `1011 0000 0001 0001` 这种二进制机器码。**汇编语言**用 `MOV AL, 17D` 让代码可读一些，但依然繁琐。**高级语言**（如 C++）则让编程变得高效：

```cpp
sum = 17 + 20;   // 一行代码等效于好几条机器指令
```

C++ 是**编译型语言**，源代码需要经过编译器转换为机器码才能运行。现代 C++（C++11 ~ C++23）每年都在进化，变得更安全、更简洁。

## 1.2 你的第一个现代 C++ 程序

### 1.2.1 最简单的程序

```cpp
int main() {
    return 0;
}
```

每个 C++ 程序都必须有一个 `main` 函数，程序从这里开始。

### 1.2.2 使用 `std::println` 输出（C++23）

传统的 `std::cout << "Hello" << std::endl;` 写起来有点啰嗦。C++23 引入了更现代的 `std::println`，自动换行，支持格式化：

```cpp
#include <print>

int main() {
    std::println("Hello, Modern C++!");
    int age = 20;
    std::println("我今年 {} 岁了。", age);
}
```

**输出**：
```
Hello, Modern C++!
我今年 20 岁了。
```

> 完整版第1章会详细对比 `std::cout` 与 `std::println` 的优缺点，并教你如何在不支持 C++23 的旧编译器上兼容。

### 1.2.3 变量：让程序不再“硬编码”

没有变量时，计算圆面积只能写死半径：
```cpp
std::println("{}", 3.14159 * 2.5 * 2.5);
```

有了变量，我们可以轻松修改半径：
```cpp
double r = 2.5;
constexpr double PI = 3.14159;
double area = PI * r * r;
std::println("面积 = {}", area);
```

`constexpr` 是定义常量的现代方式，比 `#define` 更安全。

### 1.2.4 从键盘读取输入：`cin`

结合 `std::cin`，程序可以接收用户实时输入：

```cpp
#include <print>

int main() {
    constexpr double PI = 3.14159;
    double radius;

    std::print("请输入圆的半径：");
    std::cin >> radius;

    double area = PI * radius * radius;
    std::println("圆的面积为：{:.2f}", area);
}
```

运行示例：
```
请输入圆的半径：2.5
圆的面积为：19.63
```

**这就是一个完整、有用的程序** —— 它不再是固定的数字，而是能与用户交互。
