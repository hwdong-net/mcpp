## 5.1 引用：变量的“别名”

引用是已存在变量的别名，操作引用就是操作原变量。

```cpp
int ival = 1024;
int& ref = ival;      // ref 是 ival 的引用
ref = 2048;           // 相当于 ival = 2048
std::println("ival = {}", ival);  // 输出 2048
```

**引用的关键特性**：
- 必须初始化（不能有空引用）。
- 一旦绑定，不可重新绑定到其他变量。
- 类型必须严格匹配。
- 不占用独立内存（只是别名）。

引用常用于函数参数（避免拷贝）和返回值，比指针更安全、简洁。

## 5.2 指针：直接操控内存地址

指针存储变量的内存地址，通过解引用 `*` 访问目标。

```cpp
int ival = 42;
int* p = &ival;       // p 指向 ival
*p = 100;             // 修改 ival 为 100
std::println("{}", ival);   // 输出 100
```

**指针的运算**：指针可以加减整数，偏移单位由所指类型大小决定。

```cpp
int arr[] = {10, 20, 30, 40};
int* p = arr;         // 指向 arr[0]
p++;                  // 现在指向 arr[1]
std::println("{}", *p);   // 输出 20
```

**空指针**：使用 `nullptr`（C++11）表示不指向任何对象。

## 5.3 数组：连续内存的容器

数组是相同类型元素的连续集合，大小必须在编译时确定。

```cpp
int a[5] = {1, 2, 3, 4, 5};
a[2] = 99;            // 修改第三个元素
for (int i = 0; i < 5; ++i)
    std::print("{} ", a[i]);   // 输出 1 2 99 4 5
```

**数组名即指针**：数组名可隐式转换为指向首元素的指针。

```cpp
int* p = a;           // 等价于 int* p = &a[0]
```

**范围 `for` 循环**（C++11）：安全遍历数组。

```cpp
for (int x : a)
    std::print("{} ", x);
```

## 5.4 动态内存分配：`new` 和 `delete`

静态数组大小固定，无法在运行时改变。动态内存让你在堆上分配灵活大小的内存。

```cpp
int n;
std::cin >> n;
int* arr = new int[n];   // 动态分配 n 个 int
for (int i = 0; i < n; ++i)
    arr[i] = i * 10;
delete[] arr;            // 必须手动释放
```

**注意**：
- 必须配对：`new` 对应 `delete`，`new[]` 对应 `delete[]`。
- 忘记释放会导致**内存泄漏**。
- 现代 C++ 推荐使用智能指针（`std::unique_ptr`）或容器（`std::vector`）代替裸指针。

> 完整版会用图示展示堆与栈的区别，演示动态分配二维数组的两种方法（指向数组的指针 vs 一维模拟），并给出避免内存泄漏的最佳实践。

## 5.5 查找与排序实战

### 5.5.1 顺序查找（O(n)）
```cpp
int a[] = {12, 46, 25, 43, 7};
int key = 25;
bool found = false;
for (int x : a) {
    if (x == key) { found = true; break; }
}
```

### 5.5.2 二分查找（O(log n)，要求有序）
```cpp
int low = 0, high = n - 1;
while (low <= high) {
    int mid = (low + high) / 2;
    if (a[mid] == key) { found = true; break; }
    else if (key < a[mid]) high = mid - 1;
    else low = mid + 1;
}
```

### 5.5.3 冒泡排序（O(n²)）
```cpp
for (int i = n-1; i > 0; --i) {
    bool swapped = false;
    for (int j = 0; j < i; ++j)
        if (a[j] > a[j+1]) {
            std::swap(a[j], a[j+1]);
            swapped = true;
        }
    if (!swapped) break;
}
```

## 5.6 项目实战：学生成绩管理系统（动态扩容）

这是一个综合练习，运用动态数组实现增删改查，并支持自动扩容。

**核心数据结构**：
```cpp
int n = 0;                // 当前学生数
int capacity = 2;         // 当前容量
double* scores = new double[capacity * 4];  // 每个学生4项成绩
```

**扩容逻辑**：
```cpp
if (n >= capacity) {
    capacity *= 2;
    double* newScores = new double[capacity * 4];
    // 复制旧数据
    for (int i = 0; i < n; ++i)
        for (int j = 0; j < 4; ++j)
            newScores[i*4 + j] = scores[i*4 + j];
    delete[] scores;
    scores = newScores;
}
```

系统提供菜单：插入、删除、修改、查询单个、显示所有、退出。
